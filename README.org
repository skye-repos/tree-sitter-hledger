* Tree Sitter for Hledger

A basic tree-sitter grammar file for hledger. Currently supported

- Journal Entries with
  - Dates
  - Status
  - Codes
  - Descriptions
    - Notes
  - Transactions with
    - Accounts & Sub-accounts
    - Currencies
    - Amounts w/ decimals
- Comments
  - Inline
  - Multi-line
